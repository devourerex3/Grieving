# Grieving

Grieving is a Unity game exploring the eerie and atmospheric concept of liminal places — those in-between spaces that evoke a feeling of unease, nostalgia, and mystery.

# Unity Version

This project was developed using Unity version **6000.0.51f1**.

If Unity Hub doesn’t automatically recognize this version, please copy and paste the following link directly into your browser: 
`unityhub://6000.0.51f1/01c3ff5872c5`

# Getting Started

To clone this project from Git, run the following command:

```bash
git clone https://github.com/devourerex3/Grieving.git
````

After cloning, open the project in Unity using Unity Hub with the specified version above.
